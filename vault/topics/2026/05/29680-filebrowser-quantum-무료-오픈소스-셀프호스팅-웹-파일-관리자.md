## 본문
- 기존 FileBrowser 오픈소스 프로젝트를 **대규모 포크**하여 고급 기능을 대폭 추가한 셀프호스팅 웹 파일 매니저  
- **OIDC, LDAP, JWT, 비밀번호+2FA, 프록시** 등 다양한 인증 방식 지원, 디렉토리 수준의 접근 제어를 사용자·그룹 단위로 설정 가능  
- SQLite 기반 **초고효율 인덱싱**으로 타이핑 중 실시간 검색 결과 표시, 파일·폴더 크기 및 다양한 필터 검색 지원  
- **다중 소스 동시 연결** 가능하며, include/exclude 규칙 적용과 `config.yaml` 기반 간소화된 설정 방식  
- Office 문서, 비디오, 앨범 아트워크, **3D 모델 썸네일** 미리보기 포함한 향상된 리스팅 브라우징  
- 공유 옵션이 세분화되어 **만료 시간, 접근 사용자(익명 포함), 테마, 파일 권한** 등을 개별 설정 가능  
- 장기 유효 **API 토큰 생성**과 `/swagger` 엔드포인트를 통한 개발자 API 문서 제공  
- UI는 멀티 액션 버튼, 검색 바, 뷰 전환 토글의 **3가지 컴포넌트 내비게이션** 구조  
- Nextcloud, Filestash, Google Drive, FileRun 등과 비교 시 **무료·오픈소스이면서 독립 바이너리 실행 가능**한 점이 핵심 차별점  
- Docker 이미지 크기 **180MB**(ffmpeg 포함), 최소 메모리 512MB, Linux·Windows·Mac 크로스 플랫폼 지원  
- shell 명령어는 **완전 제거**되었으며 복원 계획 없음  
- "**Quantum**"이라는 이름은 **수많은 고급 기능을 작고 실행하기 쉬운 파일 하나에** 담았다는 의미  
- 최고의 오픈소스 셀프호스팅 파일 브라우징 애플리케이션을 목표로 하며, **영구 무료·오픈소스**를 유지할 방침  
- Apache-2.0 라이선스
