## 본문
- 작업을 **퀘스트(quest) 단위**로 쪼개고 진행 상황을 추적해, 사이드 프로젝트가 흐지부지되는 패턴을 막아주는 데스크톱 앱  
  - 제작자가 GitHub 레포 134개를 만들었지만 끝맺지 못한 경험에서 출발, **의지력만으로 부족할 때를 위한 도구**로 개발  
- **퀘스트 시스템**: 단계가 순서대로 잠금 해제되는 구조이며, 수동으로 편집하거나 AI로 트리를 생성한 뒤 수정 가능  
- **스트릭(streak) 트래킹**: 일일 연속 진행 카운터와 마일스톤 제공, 공개 스코어보드가 아닌 개인용 기록  
- **재진입 요약(Re-entry summaries)**: 어디서 멈췄고 다음에 무엇을 할지 AI가 선택적으로 요약, 이전 노트를 다시 읽을 필요 없음  
- **범용 활용**: 프로그래밍, 학습, 시험 준비, 글쓰기, 디자인, 언어, 피트니스 등 단계로 쪼갤 수 있는 모든 영역에 사용 가능  
- **로컬 우선 설계**: 계정 필요 없음, 오프라인 동작, 모든 데이터는 로컬 기기에 저장  
- **자체 AI 키(BYOK)**: Anthropic 또는 OpenAI 키를 직접 등록해 플랜 초안 생성, 수동 모드에서는 키 불필요, 키는 기기를 벗어나지 않음  
- v1.0.9 부터 **OpenAI 호환 API 커스텀 엔드포인트** 추가, Ollama, Groq, OpenRouter, LM Studio, Gemini 호환 등과 연동 가능  
  - 내장 모델에 최신 Anthropic 모델과 OpenAI 추론 모델(o1, o1-mini, o3-mini, o4-mini) 추가  
  - Anthropic / OpenAI / Other 섹션으로 그룹화된 모델 선택기 제공  
- 무료, 구독 없음, macOS / Windows / Linux 지원
