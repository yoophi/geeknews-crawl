## 본문
- **Unified** 디자인이 도입되어 탭, 도구 모음, 패널, 콘텐츠의 경계를 줄이고 브라우저 전체를 하나의 연속된 프레임으로 구성함
- 모든 도구 모음이 단일 표면 위에 놓이며, 테마 색상과 배경이 탭 바부터 패널과 가장자리까지 더 일관되게 이어짐
- 배경화면은 더 이상 일부 영역에만 갇히지 않고, **투명도**와 블러 효과와 함께 브라우저 환경의 일부처럼 동작할 수 있음
- 새 기본 테마로 **Zen**, **Soria Moria**, **Sunset Forest**, **Kawaii Clouds**가 추가되고, 기존 라이트/다크 테마도 업데이트됨
- 기본 테마가 맞지 않으면 [themes.vivaldi.net](http://themes.vivaldi.net/)에서 7000개 이상의 테마를 고를 수 있음
- 기존 커스텀 테마 사용자는 `Settings → Themes → Editor → Coloring mode`에서 새 디자인을 쓸지, 기존 모양을 유지할지 선택 가능함
- 온보딩과 `Settings → Appearance → Layout`에서 **Simple**, **Classic**, **Vertical Right**, **Vertical Left**, **Auto Hide**, **Bottom** 6가지 레이아웃 프리셋을 바로 선택할 수 있음
- **Auto Hide** 레이아웃은 도구 모음, 탭 바, 패널, 주소 표시줄을 숨겼다가 커서가 화면 가장자리로 이동하면 다시 표시해 콘텐츠 영역을 넓게 쓸 수 있게 함
- **Vertical Left**는 탭과 주소 표시줄을 화면 왼쪽에 배치하고, 패널은 오른쪽에서 호버로 사용할 수 있어 넓은 모니터의 가로 공간 활용을 바꿈
- **Bottom** 레이아웃은 탭 바와 주소 표시줄을 화면 아래로 옮겨, 읽는 흐름에 맞춰 내비게이션을 아래쪽에 배치함
- Vivaldi의 탭 관리 기능은 페이지 타일링, Follower Tab, 이름과 색상이 있는 탭 스택, Tab Button의 열린 탭·동기화 탭·최근 닫은 탭 검색을 계속 제공함
- 전체 변경 목록은 [Vivaldi 7.9에서 8.0까지의 changelog](https://vivaldi.com/changelog-vivaldi-browser-8-0)에서 확인 가능함
