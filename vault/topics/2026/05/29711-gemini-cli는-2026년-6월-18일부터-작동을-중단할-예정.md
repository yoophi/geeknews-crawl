## 본문
- **Gemini CLI**는 터미널에서 Gemini를 쓰는 도구로 성장했지만, Google은 멀티 에이전트 흐름에 맞춰 Antigravity CLI로 역량을 통합함
- Gemini CLI는 수백만 사용자, GitHub 스타 10만 개 이상, 병합된 풀리퀘스트 6,000개와 **수백 명의 기여자**를 확보함
- **Antigravity CLI**는 Go 기반 새 터미널 경험과 서버 측 하네스로 여러 에이전트의 백그라운드 작업을 오케스트레이션함
- 초기에는 1:1 기능 동등성이 없지만 **Agent Skills**, Hooks, Subagents, Extensions의 핵심 기능은 유지됨
- 2026년 6월 18일부터 무료·Pro·Ultra 사용자의 Gemini CLI와 Gemini Code Assist IDE extensions 요청 처리가 중단되며, **엔터프라이즈 접근 권한**은 유지됨

### Gemini CLI에서 Antigravity CLI로 전환
- Gemini CLI는 터미널에서 Gemini를 직접 쓰는 도구로 출시됐고, 수백만 사용자, **GitHub 스타 10만 개 이상**, 병합된 풀리퀘스트 6,000개, 수백 명의 기여자를 확보함
- 사용자 워크플로는 2025년 초기 단계를 넘어 복잡한 문제를 나눠 풀기 위해 **여러 에이전트가 서로 통신**해야 하는 방향으로 바뀜
- Google은 멀티 에이전트 흐름에 맞춘 단일 제품에 집중하기 위해 역량을 [Google Antigravity](http://antigravity.google/blog/introducing-google-antigravity-2-0)로 통합함
- [Antigravity CLI](http://antigravity.google/blog/introducing-google-antigravity-cli)는 강력한 서버 측 하네스와 새로운 터미널 경험을 포함하는 Antigravity의 CLI임
- 초기에는 Gemini CLI와 **1:1 기능 동등성**이 없지만, Agent Skills, Hooks, Subagents, Extensions는 핵심 기능으로 유지됨
- Extensions는 Antigravity에서 **Antigravity plugins** 형태로 제공됨
- 빠른 근거 기반 답변, 새 코딩 프로젝트의 스캐폴딩과 구현, 클라우드 인프라 프로비저닝 지원은 Antigravity CLI에서도 가능함

### 일정과 접근 권한 변화
- ## 기능과 구조
  - Antigravity CLI는 **Go로 구현**돼 더 빠르고 반응성이 좋은 실행 경험을 제공함
  - **비동기 워크플로**를 지원해 복잡한 작업을 여러 에이전트가 백그라운드에서 오케스트레이션함
  - 대규모 리팩터링이나 여러 주제 조사를 실행해도 터미널 세션이 잠기지 않음
  - [Antigravity 2.0](https://antigravity.google/blog/introducing-google-antigravity-2-0) 및 새 Antigravity 데스크톱 앱과 같은 **에이전트 하네스**를 공유함
  - 핵심 에이전트 개선 사항은 사용하는 위치와 관계없이 자동으로 적용되는 구조임
- ## 소비자 대상 일정
  - [Antigravity CLI](https://antigravity.google/download)는 2026년 5월 19일부터 모두에게 제공됨
  - 2026년 6월 18일에 Gemini CLI와 Gemini Code Assist IDE extensions는 Google AI Pro, Ultra, 무료 사용자를 대상으로 요청 처리를 중단함
  - 무료 사용자에는 [Gemini Code Assist for individuals](https://developers.google.com/gemini-code-assist/docs/overview#supported-features-gca) 사용자가 포함됨
  - 전환을 돕기 위해 [기술 문서](http://antigravity.google/docs/gcli-migration)가 제공되며, 향후 몇 주 안에 동영상 안내도 공개될 예정임
  - [Gemini Code Assist for GitHub](https://developers.google.com/gemini-code-assist/docs/review-repo-code)에도 같은 변경이 적용되며, 2026년 6월 18일부터 GitHub 조직에 새로 설치할 수 없음
  - Gemini Code Assist for GitHub 요청은 그 뒤 몇 주 동안 중단될 예정임
- ## 엔터프라이즈 고객
  - 조직이 Gemini Code Assist Standard 또는 Enterprise 라이선스로 Gemini CLI나 IDE 확장을 사용하는 경우 접근 권한은 바뀌지 않음
  - 조직이 Google Cloud를 통해 Gemini Code Assist for GitHub를 사용하는 경우에도 접근 권한은 유지됨
  - Gemini CLI와 Gemini Code Assist는 최신 Gemini 모델 접근과 기타 업데이트를 계속 지원받음
  - Gemini CLI는 유료 Gemini 및 Gemini Enterprise Agent Platform API 키로 계속 접근 가능함
  - Antigravity CLI는 [Google Cloud 프로젝트](http://antigravity.google/blog/google-antigravity-for-enterprises)와 함께 바로 사용할 수 있음
- ## 전환 지원
  - 기능 요청과 전환 관련 피드백은 Antigravity CLI [커뮤니티 포럼](https://github.com/google-antigravity/antigravity-cli)에서 받음
  - Google I/O 2026 업데이트와 발표 내용은 [io.google](https://io.google/2026/?utm_source=blogpost&utm_medium=pr&utm_campaign=devblogs&utm_content=)에서 확인 가능함
